# Sprint 2 Report (03/02/2026 - 04/04/2026)
## What's New (User Facing)
* Added a navigation bar for simple user navigation.
* Separated login and register page
* Implemented email verification 
* Preferences and password reset connected to database
* Creation of Pins
* Existing Library of Building Pins
* Private Pins and Events
* Feature n or Bug Fix n
## Work Summary (Developer Facing)
This sprint we continue to add new features for our campus app. We implemented an email verification feature to send verification codes to a users email. We also added our first implementation of pins so that users can add custom pins to the map and see existing building pins. Navigation was also improved allowing users to seamlessly transition between pages using a navigation bar with custom icons.
## Unfinished Work
This sprint we did not incorporate building intractability or gps integration due to lack of time and lesser importance. These features will be completed in the next sprint. Progress was made on the final UML diagram, but completion will be delayed until the final app design is realized. Schedule companion app was also not worked on. This feature is low priority and may be worked on next sprint otherwise it may be scaled back.
